# 🎓 SkillSphere - Modern Online Learning Platform

**SkillSphere** is a high-performance, interactive learning platform designed to help users enhance their skills. It features a clean, futuristic techno-aesthetic interface that bridges frontend architecture with premium UI design.

## 🚀 Live Demo
🔗 **Live URL:** [https://skill-sphere-learn.vercel.app/](https://skill-sphere-learn.vercel.app/)

---

## ✨ Key Features
*   **Responsive Design:** Fully optimized for mobile, tablet, and desktop devices.
*   **Secure Authentication:** Fast and secure sign-in via Google Social Login powered by Better Auth.
*   **Interactive Course Listings:** Dynamic curriculum displays with a focus on modern UI/UX trends like glassmorphism.
*   **User Profiles:** Users can manage their personal information and track their real-time online status.
*   **Modern Navigation:** Features a specialized mobile hamburger menu for seamless navigation on smaller screens.

---

## 🛠 Technology Stack
This project is built using a modern full-stack development environment:

*   **Frontend:** React & Next.js
*   **Styling:** Tailwind CSS with custom glassmorphism effects
*   **Authentication:** Better Auth (Google Provider)
*   **Database:** MongoDB
*   **Deployment:** Vercel
